    ベーシック認証Link

    conf fileLink

      Satisfy Any
  Order Deny,Allow
  Deny from all
  Allow from i.p.add.r
  AuthType Basic
  AuthName "Please enter your Id and Password."
  AuthUserFile /var/www/vhosts/.htpasswd
  Require valid-user
